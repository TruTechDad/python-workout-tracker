# Workout Tracker

[![License: MIT](https://img.shields.io/badge/license-MIT-blue)](https://opensource.org/licenses/MIT)

## Description

A lightweight Python workout tracker built to log exercises, sets, reps, and workout sessions directly from the command line. This project focuses on practical Python development, clean data handling, and creating a simple but functional fitness tracking tool for monitoring workout progress.
